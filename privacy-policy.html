개인정보처리방침

시행일: 2026년 7월 2일

kids-play(이하 "앱")는 사용자의 개인정보를 중요하게 생각합니다. 본 개인정보처리방침은 앱 사용 시 수집되는 정보와 그 처리 방법에 대해 설명합니다.

1. 수집하는 정보

본 앱은 개인정보를 수집하지 않습니다. 앱은 웹뷰를 통해 외부 웹사이트를 표시하며, 별도의 개인정보를 수집하거나 저장하지 않습니다.

2. 인터넷 사용

앱은 콘텐츠를 로드하기 위해 인터넷 연결이 필요합니다. 인터넷 사용 과정에서 발생하는 데이터는 앱이 수집하거나 저장하지 않습니다.

3. 제3자 서비스

앱은 GitHub Pages에서 호스팅되는 웹 콘텐츠를 표시합니다. 해당 콘텐츠의 개인정보 처리는 GitHub의 개인정보처리방침을 따릅니다.

4. 아동 개인정보 보호

본 앱은 아동을 위한 서비스로, 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

5. 개인정보처리방침 변경

개인정보처리방침이 변경될 경우 이 페이지를 통해 고지합니다.
